dynks
=====

dynks [dɨ̃ŋks]:
    Polish noun; an item with a name that the speaker cannot determine.

**dynks** is a collection of various Python utilities that make software
developer's life easier.

Installation
------------

Using ``pip``:

.. code-block:: bash

    pip install dynks

Using ``pipenv``:

.. code-block:: bash

    pipenv install dynks
